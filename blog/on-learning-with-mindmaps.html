On Learning with Mind-maps

January 2026

My note-taking process starts with structure, not details. I organize content in a broad mind-map before I go deep into the material.

My Process

  1. I skim what I am reading and broadly organize the content in a mind-map before I go deep into the material.
  2. I then read the resource. I occasionally add to the map, but most of this phase is deep reading with minimal note-taking.
  3. When my understanding changes enough, I rewrite the mind-map from scratch. This usually happens multiple times when learning any hard subject.
  4. I free-recall constantly, including dedicated recall sessions before I start the next batch of content.

Important Note

This is what works for me. It is not a universal recommendation. I arrived at this process through a lot of trial and error.

Recommendations

  • Aim for higher-order learning. Bloom's revised taxonomy is a useful guide.
  • Do not write excessively. Writing is a tool to support thinking; thinking is what actually drives learning.
  • Try mind-mapping seriously. It is a skill that takes practice, but it can radically improve how fast and how deeply you learn.

Every person is different. The key is to test methods honestly and keep what actually works for your own learning.
